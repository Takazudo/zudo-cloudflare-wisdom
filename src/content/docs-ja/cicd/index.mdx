すべてのプロジェクトで GitHub Actions を使って Cloudflare にデプロイしています。このセクションでは標準的なワークフローパターンを解説します。

## 標準パイプライン

```mermaid
graph LR
  A[Push to main] --> B[Build]
  B --> C[Deploy to CF Pages]
  C --> D[IFTTT Notification]
```

PR の場合：

```mermaid
graph LR
  A[Open PR] --> B[Type Check]
  A --> C[Build]
  C --> D[Preview Deploy]
  D --> E[Comment URL on PR]
```
